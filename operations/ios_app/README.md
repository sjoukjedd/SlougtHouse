# VUAMS iOS App

## Xcode project aanmaken

1. Installeer XcodeGen: `brew install xcodegen`
2. Navigeer naar deze map: `cd operations/ios_app`
3. Genereer project: `xcodegen generate`
4. Open: `open VUAMS.xcodeproj`

Vereisten: Xcode 15+, iOS 17+ device of simulator

> **Let op:** Vul `DEVELOPMENT_TEAM` in `project.yml` in met je Apple Developer Team ID voor signing.
